FAQ
Apa itu FlowGram?+
FlowGram adalah workflow builder visual untuk membuat, menghubungkan, dan mengelola alur kerja di kanvas tanpa batas.
Bagaimana cara login?+
Klik "Masuk dengan Google" di halaman awal, pilih akun Google kamu, dan selesai.
Apa itu demo mode?+
Demo mode memungkinkan kamu mencoba FlowGram tanpa login. Data disimpan di localStorage browser kamu.
Bagaimana cara migrasi dari demo ke akun?+
Klik "Sign in here" di sidebar (bawah), login dengan Google, data kamu akan otomatis dipindahkan ke cloud.
Apakah data saya aman?+
Ya. Data tersimpan di database Neon PostgreSQL dan hanya bisa diakses oleh akun Google kamu.
Bagaimana cara menghubungkan node?+
Klik dan tarik dari titik bulat (connector dot) di tepi node ke node lain. Garis kurva akan otomatis terhubung.
Bagaimana cara menyisipkan icon di node?+
Klik kanan pada node, pilih "Pilih Icon", lalu pilih icon dari 6 kategori yang tersedia.
Bagaimana cara mengubah warna node?+
Klik kanan pada node, pilih "Warna", lalu pilih dari 8 warna (default, merah, oranye, kuning, hijau, biru, ungu, pink).
Bagaimana cara import/export file .json?+
Klik "Export" di toolbar untuk menyimpan workflow sebagai file .json. Klik "Import" untuk memuat file .json yang sudah disimpan sebelumnya.
Bagaimana cara mengganti font?+
Buka Settings > General > Personalize, pilih font favorit kamu dari 9 pilihan yang tersedia.
Bagaimana cara mengekspor semua project?+
Buka Settings > Privacy, klik "Ekspor Semua Data".
Bagaimana cara menghapus akun?+
Buka Settings > Account, klik "Delete account". Semua data akan dihapus permanen.
Apakah FlowGram punya dark mode?+
Ya. Atur di Settings > General > Appearance, atau ikuti preferensi sistem kamu.
Apakah FlowGram bisa di HP?+
Ya. FlowGram responsif di semua ukuran layar dan mendukung sentuhan, pinch-to-zoom, dan bottom sheet.
About
FlowGram adalah workflow builder visual untuk membuat, menghubungkan, dan mengelola alur kerja dengan mudah.
Fitur:
• Infinite canvas — pan, zoom, atur node bebas
• Node connections — kurva bezier dengan animasi
• Multi-project dashboard — kelola banyak workflow
• Google OAuth — masuk dengan akun Google
• Neon database — project tersimpan di cloud
• Demo mode — coba tanpa login
• Font picker — pilih dari 9 jenis font
• Dark mode — tema terang dan gelap
• Mobile-friendly — sentuhan, pinch-to-zoom
Tech: Vanilla JS, Hono.js, Neon PostgreSQL, Google OAuth, Lucide Icons, Bootstrap Icons
